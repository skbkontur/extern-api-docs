Работа с учетными записями
==========================

В данном разделе описаны методы API для работы с учетными записями. С их помощью можно просмотреть список доступных учетных записей авторизованного пользователя, выбрать необходимую учетную запись для последующей работы с API, просмотреть список доступных сертификатов и организаций для работы с указанной учетной записью и т.д. Также при определенных условиях существует возможность создания новой учетной записи.
