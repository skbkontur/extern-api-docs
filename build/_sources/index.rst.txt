API Контур Экстерна
===================

.. image:: /logo.png
   :width: 550


API Контур.Экстерн позволяет интегрировать сервис отчетности в вашу информационную систему. Можно настроить свои уникальные сценарии работы с отчетностью и реализовать отправку в такие контролирующие органы, как ФНС, Росстат и СФР.
