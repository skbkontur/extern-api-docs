Примеры файлов отчетов
======================

ФНС
~~~

.. note:: Все файлы отчетов в формате xml, которые будут отправлены в ИФНС, должны быть сохранены в кодировке windows-1251.

**Бухгалтерская отчетность**

1. Декларация без ошибки.

  * :download:`пример файла бухгалтерской отчетности <../files/fns example/NO_BUHOTCH_0087_0087_6676130154667601001_20241024_39fc932f-9cf2-4344-821a-4d71167ca1e0.xml>`

.. _rst-markup-exmp2:

2. Декларация для отправки за другую организацию, ошибка при подготовке.

  * :download:`пример файла бухгалтерской отчетности <../files/fns example/NO_BUHOTCH_0087_0087_6676130154667601001_20241024_8683274e-03a8-4054-a12b-959b29056600.xml>`
    
  * :download:`пример сообщения о представительстве <../files/fns example/ON_DOV_0087_0087_6676130154667601001_20241024_d075a2c7-1e38-49f1-a13b-28753780102c.xml>`

.. _rst-markup-exmp3:

3. Декларация с некорректными документами для проверки работы Check.


  * :download:`пример файла бухгалтерской отчетности с ошибкой <../files/fns example/NO_BUHOTCH_0087_0087_6676130154667601001_20241024_04621bd7-14b3-4c78-ab67-ddbf91cc76d6.xml>`

  * :download:`пример сообщения о представительстве <../files/fns example/ON_DOV_0087_0087_6676130154667601001_20241024_d075a2c7-1e38-49f1-a13b-28753780102c.xml>`
    
  * Приложение (пояснительная записка).

    Можно взять любой файл *.pdf, и присвоить ему имя, например:

    ::

      NO_BUHPZ_0087_0087_6676130154667601001_15b810f1-6a58-4d17-8185-90336d5005b4_20180411_15b810f1-6a58-4d17-8185-90336d5005b4.pdf

**Налог на прибыль**

:download:`пример файла декларации Налог на прибыль <../files/fns example/NO_PRIB_0007_0007_6676130154667601001_20241024_9c79ed50-87af-4925-8e07-ed3404a13d8d.xml>`
  
Данный файл уже сохранен в кодировке windows-1251. Если используете его для отправки, убедитесь, что:

  * ФИО в теге Подписант совпадает с ФИО в сертификате, которым будет подписан документооборот,
  * НПЮЛ совпадает с данными в сертификате, которым будет подписан документооборот.
